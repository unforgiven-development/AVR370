MEGA-1284P Xplained board

Introduction

This example demonstrates some of the basic features of the MEGA-1284P Xplained kit. This includes sleep modes, reading the light sensor and the ntc sensor.

Brief description of the example application

The example application have two main modes: demo mode or terminal mode. Please refer to the detailed description for more informaiton.
